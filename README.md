# Pattachitra Art Website

## Project Overview

The **Pattachitra Art Website** is a multi-page web project created to present the traditional Indian painting style *Pattachitra*. The website introduces the art form, displays different paintings, and allows users to inquire about purchasing or commissioning artworks.

The main goal of this project is to create a simple and elegant website that showcases traditional art using modern web design.

---

## Features Implemented

* Multi-page website design
* Navigation bar for easy access to all pages
* Gallery page with category filter (All, Mythology, Nature, Contemporary)
* Image hover zoom effect in the gallery
* Shop page with artwork cards showing image, size, and price
* Inquiry button with popup message
* Commission section for custom artwork requests
* Contact page with studio details and contact form

---

## Pages in the Website

### Home Page
Provides an introduction to Pattachitra art and the purpose of the website.

### About Page
Explains the history, origin, and cultural significance of Pattachitra paintings.

### Gallery Page
Displays a collection of paintings in a grid layout with filter options.

### Shop Page
Shows selected artworks with details like size and price. Users can click the **Inquire** button to express interest in a painting.

### Contact Page
Allows users to contact the artist through email, phone, or a contact form.
